# AutoChess - AI vs AI Infinite Match

A fully automatic, self-playing chess game built with HTML, CSS, and JavaScript. Watch two AI bots battle endlessly with premium aesthetics and glassmorphism effects.

## Features
- **AI Personalities**: Bots have unique playing styles (Aggressive, Positional, Balanced)
- **Opening Detection**: Displays famous chess openings as they're played
- **Visual Evaluation Bar**: Real-time material advantage indicator
- **Captured Pieces Display**: See what each side has captured
- **Sound Effects**: Audio feedback for moves, captures, and checks
- **Speed Control**: Toggle between Normal, Fast, and Slow speeds
- **Fully Responsive**: Works seamlessly on desktop and mobile
- **No Server Required**: Pure client-side, runs on GitHub Pages

## Live Demo
Visit: [automaticchess.github.io](https://automaticchess.github.io)

## Tech Stack
- Pure HTML/CSS/JavaScript
- chess.js library for game logic
- Web Audio API for sound effects
- No external dependencies or build tools

## Controls
- **New Game**: Start a fresh match
- **Speed**: Cycle through game speeds
- **Sound**: Toggle audio on/off
