# AutoChess - AI vs AI Infinite Match

A fully automatic, self-playing chess game built with HTML, CSS, and JavaScript. Designed with premium aesthetics and glassmorphism effects.

## Features
- **Automatic Gameplay**: Two AI bots play against each other endlessly.
- **Premium UI**: Glassmorphism design, neon glow effects, and smooth animations.
- **Responsive**: Works on desktop and mobile.
- **No Dependencies**: All libraries are included locally.

## Customization
- **Speed**: Click the "Speed" button to toggle between Normal, Fast, and Slow.
- **Reset**: Click "New Game" to restart the match.
